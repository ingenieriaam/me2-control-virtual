## En referencia al entorno grafico 
Del html, se describe cada comando necesario.

_______________________________

> set duty cycle

```sh
CWDUTY<canal>,<duty_cycle>
```

> duty on off

```sh
CWDUTYS <canal>,<state>
```

> average

```sh
CWAVG canal, ON/OFF
```

> promedios: unico que tenia ejemplos

```sh
CWAVG canal, MOV, 32
```

> max min ????????

